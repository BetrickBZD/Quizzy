# Quizzy 🧠✨

Willkommen bei **Quizzy**! Eine interaktive und unterhaltsame Quiz-App, mit der Nutzer ihr Wissen in verschiedenen Kategorien testen und erweitern können. 

![Quizzy Banner/Screenshot](Link-zu-deinem-Bild-oder-Screenshot.png) ## 🚀 Features

* **Vielfältige Kategorien:** Wähle aus verschiedenen Themenbereichen (z. B. Allgemeinwissen, Technologie, Geschichte).
* **Echtzeit-Feedback:** Sofortige Auswertung der Antworten nach jeder Frage.
* **Highscore-Tracking:** Behalte deine besten Ergebnisse im Blick.
* **Responsive Design:** Sieht auf dem Desktop, Tablet und Smartphone großartig aus.
* **Benutzerfreundliches Interface:** Intuitive Bedienung für ein reibungsloses Spielerlebnis.
